Expex

  • Expex is a character code to character converter.
  • Type your number and press enter.
  • Type in a single number and the corresponding character will be displayed.
  • Type in two numbers separated by a dash(-) and the corresponding range of characters will be displayed.
  • Type in numbers separated by commas(,) and the corresponding character for each number will be displayed.
  • Tip: Use the generated characters on your webpage by preceding the char code with '&#' and succeeding it with ';'. Eg. - &#10006;
  • Use Alt+A to focus on the text box.
  • Use Alt+R to try a random 4-digit (or less) number.
  • Use Alt+H to view history.
